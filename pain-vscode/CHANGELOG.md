# Change Log

All notable changes to the Pain Language extension will be documented in this file.

## [0.2.0] - 2024-01-XX

### Added
- Format document command (`pain.formatDocument`)
- Format on save support (configurable via `pain.format.enable`)
- Enhanced completion with break, continue, return keywords
- Better completion details for built-in functions

### Changed
- Updated LSP server integration
- Improved error handling

## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Syntax highlighting for Pain language
- LSP integration (diagnostics, completion, hover)
- Language configuration (comments, brackets, etc.)
